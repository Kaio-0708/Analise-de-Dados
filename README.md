#  Análise de Dados 

Este repositório tem como objetivo **armazenar projetos de aprendizado em Análise de Dados**, desenvolvidos com **Python** e as principais **ferramentas utilizadas na área**.

Cada pasta dentro deste repositório representa um projeto independente, com seus próprios scripts, notebooks, datasets e resultados.

---

##  Objetivo do Repositório

- Praticar conceitos de **análise exploratória de dados (EDA)**  
- Utilizar **bibliotecas populares do ecossistema Python**  
- Aprender técnicas de **limpeza, transformação e visualização de dados**  
- Criar relatórios e dashboards interativos

---

##  Ferramentas Fundamentais

As principais bibliotecas e ferramentas utilizadas neste repositório são:

| Ferramenta | Função |
|-------------|--------|
| **Pandas** | Manipulação e limpeza de dados |
| **NumPy** | Operações matemáticas e vetorização |
| **Matplotlib** | Visualização básica de gráficos |
| **Seaborn** | Visualização estatística (mais bonita e fácil que o Matplotlib) |
| **Plotly** | Dashboards e gráficos interativos |
| **Scikit-learn** | Aprendizado de máquina básico e estatística |
| **Jupyter Notebook** | Ambiente interativo para análises e experimentação |
